磐垒主动防御 (Bulwark) 使用说明
================================

系统要求：
- Windows 10/11（64位）
- .NET 8.0 SDK

安装步骤：
--------
1. 下载并安装 .NET 8.0 SDK
   下载地址：https://dotnet.microsoft.com/download/dotnet/8.0
   选择 "Download .NET SDK x64" 按钮

2. 解压本文件夹到任意位置

3. 右键点击"安装服务.bat"，选择"以管理员身份运行"
   等待安装完成（约1-2分钟）

4. 双击"启动UI.bat"运行程序

卸载方法：
--------
右键点击"卸载服务.bat"，选择"以管理员身份运行"

常见问题：
--------
Q: 提示"找不到 .NET SDK"
A: 请安装 .NET 8.0 SDK（不是 Runtime）

Q: 服务安装失败
A: 请确保以管理员身份运行

Q: 程序无法启动
A: 请确保已安装 .NET 8.0 SDK

Q: 如何查看服务状态？
A: 打开命令提示符(管理员)，输入 sc query BulwarkDefense
